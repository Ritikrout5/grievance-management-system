# 🎓 Online Grievance and Solution Management System

## 📌 Project Description
This is a web-based application designed to help students and faculty submit, track, and resolve grievances efficiently.

## 🚀 Features
- Submit grievances online
- Track complaint status
- Admin dashboard for resolution
- Real-time updates using Firebase

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Firebase
- Firestore

## 📂 Project Structure
- index.html → Homepage
- dashboard.html → User dashboard
- css/ → Styling files
- js/ → JavaScript logic
- firebase-config.js → Firebase setup

## 🔥 How It Works
1. User logs in
2. Submits grievance
3. Data stored in Firestore
4. Admin reviews and updates status
